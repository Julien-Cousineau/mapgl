# mapGL
WebGL wrapper

## Install

Install using NPM (`npm install @julien.cousineau/mapgl`), then:

```js
// import as a ES module
import MapGL from '@julien.cousineau/mapgl';

// or require in Node / Browserify
const MapGL = require('@julien.cousineau/mapgl');
```
